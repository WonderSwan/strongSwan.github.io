    • IKEv2 for Blackberry

    Download CACert.pem certificate. Please check your email to find CACert.pem Download Address.

    You can download it to your laptop and then place the certificates file into your SD card.

    (Another way is to open the download link in the web browser of your Blackberry phone, the certificate will be installed directly. It is much more simple.)

    Please Contact us if no order email is received after successful payment.

    On the home screen, swipe down from the top of the screen.

    Tap Settings

    In the BlackBerry® System Settings, Security and Privacy > Certificates > Import (CACert.pem)

    In the BlackBerry® System Settings, expand Networks and Connections > VPN. Click Create VPN profile.

    Profile Name: anything, this field does not matter
Server Address: Server IP
Gateway Type: Generic IKEv2 VPN Server
Authentication Type: EAP-MSCHAPv2
Authentication ID Type: IPv4
MSCHAPv2 EAP Identity: anything, this field does not matter
MSCHAPv2 Username: Username
MSCHAPv2 Password: Password
Gateway Auth Type: PKI
Gateway Auth ID Type: Identity Certificate Distinguished Name
Gateway CA Certificate *: strongSwan Root CA
Perfect Forward Secrecy: not checked
There is no need to change any 'Advanced' configurations.

    If everything goes smoothly, you should be connected to the VPN.
